# FTC 平台中长期增强列表（仓库化 / 持久化导向）

## 角色视角拆解

### 出题者（Challenge Creator）
- 题目仓库版本管理：支持题面/Flag/附件的版本记录、对比与回滚，便于长期维护与复用。
- 题目标签与搜索：按难度/分类/标签/作者快速筛选题目库；支持收藏/批量导出。
- 题目模板与复用：预置多语言/多框架题目模板，支持一键复制到比赛或题库。
- 质量评审流程：草稿/审核/发布流，允许多角色协作评审与留痕。
- 靶机镜像管理：镜像/Compose 模板仓库，支持镜像版本标记、自动过期和扫描结果存档。

### 举办方（Organizer）
- 多场景复用：同一比赛配置支持复制为“训练场”或“公开练习房”，开启/关闭记分板。
- 灵活报名/权限：私有比赛白名单、邀请码批量导入、赛道分组（解锁不同题集）。
- 成绩与审计：选手/队伍得分调整审核流，提供调整日志导出。
- 赛务自动化：赛前/赛后广播模板，赛程提醒，成绩单（PDF/CSV）导出。
- 安全与风控：可疑行为检测（异常提交频率/IP 重复/代理检测），可配置封禁与提示。
- 外部登录：OAuth2 / SSO（GitHub/GitLab/企业 IdP）对接；角色同步与最小权限映射。

### 参赛选手（Player）
- 个人练习模式：比赛结束后可将题目加入“复现清单”，支持个人靶机启动与提交回放。
- 题解与讨论：赛后题解发布、评论/点赞，管理员可审核/隐藏敏感内容。
- 进度追踪：解题历史时间线、错题本、提示花费统计；个人/队伍成就徽章。
- 通知与推送：WebSocket/邮件/站内信多渠道通知（榜单变更、公告、题解、靶机到期）。

## 技术/平台级增强
- API 安全：补齐 OAuth2/SSO 登录流、Refresh Token 轮换、设备指纹与强制下线通知。
- WebSocket：多节点 Redis 节流/鉴权联动，事件类型规范与前端 SDK 封装。
- 存储与导入导出：大附件分片上传/秒传，题目/题库全量导入导出（含镜像/附件校验）。
- 观测性：审计日志细化（权限变更、配置调整、题目版本），Prometheus 指标暴露，慢查询与异常告警。
- 多租户/分区：预留命名空间/组织隔离能力（题库、比赛、靶机、用户隔离与配额）。
- 国际化：中英双语题面与界面资源，时间/货币/地区格式自动化。
