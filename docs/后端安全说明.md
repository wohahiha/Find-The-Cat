# 后端安全标准（FTC CTF 平台）

本文档定义后端安全基线与实践要求，适用于 Django + DRF 架构的所有业务模块（accounts/contests/challenges/submissions/machines/problem_bank/system）。

## 1. 认证与授权
- **JWT 认证**：使用 `apps.common.authentication.JWTAuthentication`；默认不写入 JWT Cookie，仅使用 Authorization 头。
- **权限**：默认 `IsAuthenticated`；管理员接口使用 `IsAdmin`/BizPermission；敏感接口显式声明。
- **登录验证码**：生产环境 `ALLOW_LOGIN_WITHOUT_CAPTCHA=False`，登录必须带图形验证码。
- **登录失败锁定**：用户名/邮箱 + IP 双维度计数，5 分钟内连续 5 次失败封禁，返回 429。

## 2. 限流与防暴力
- DRF throttle 统一使用 `apps.common.throttles`：
  - login、email_code_send、flag_submit、machine_start、user_post、attachment_upload。
- 验证码校验失败计数：同邮箱+场景 5 分钟内连续 5 次错误封禁（返回 429）。
- 验证码有效期：强制 5–30 分钟区间，后台配置超出将被收敛。

## 3. 输入与输出校验
- **密码**：8–64 位，必须同时包含字母和数字。
- **邮箱/slug/URL**：使用统一校验器，URL 必须合法协议；禁止空/非法格式。
- **文本字段**：昵称/简介/组织/国家等限制长度，禁止包含 `<`/`>`（防止 XSS 注入）。
- **富文本/展示内容**：前后端渲染时必须转义或使用白名单渲染（需在具体页面实现）。

## 4. 文件上传安全
- 统一使用 `validate_upload_file` / `validate_image_file` 校验 MIME、后缀、大小：
  - Logo：PNG/JPEG/WEBP，≤2MB。
  - 题目附件：zip/tar/gz/bz2/7z/txt/pdf/md/json 等，≤10MB。
  - 题库导入：zip/tar/gz/bz2/7z，≤50MB。
- 存储：通过 `apps.common.infra.file_storage` 统一存储路径，禁止路径穿越；必要时接入对象存储与 AV 扫描。

## 5. 邮件/验证码
- 发信账号统一走 `MailAccount`；from_email 固定等于用户名。
- 验证码邮件：主题/有效期可配置，Logo 内联 CID 避免外链失败。
- 失败收件人抛 ValidationError，其他异常抛 EmailSendError，避免泄露底层栈。

## 6. 安全日志与审计
- 安全事件 logger：`apps.security`（归入系统日志）。
  - 事件：login_success / login_failed / send_email_code。
  - 记录：用户名、user_id、IP、路径、request_id、失败原因/场景等。
- 管理员操作/系统日志：保留 AdminActionLog/SystemLog 导出。

## 7. 配置与部署
- **必须**：生产环境 DEBUG=False，SECRET_KEY 非默认；ALLOWED_HOSTS 配置实际域名；HTTPS 强制。
- Web 服务器暴露 `/static/` 和 `/media/`；日志/上传目录需正确权限。
- CORS：生产仅允许受信来源；禁止 `ALLOW_LOGIN_WITHOUT_CAPTCHA` 在生产开启。
- 邮件/DB/Redis 等敏感配置通过环境/后台提供，避免硬编码默认。

## 8. 依赖与补丁
- 定期升级依赖，关注 Django/DRF/第三方组件安全公告。
- 启用常见 HTTP 安全头（HSTS/CSP/Referrer-Policy 等）由 Web 服务器或中间件配置。

## 9. 安全测试基线
- 用例覆盖：
  - 登录失败锁定、验证码限流/失败封禁、生产需验证码。
  - 上传类型/大小校验；Logo 类型拒绝；XSS 输入拒绝（HTML 标签）。
  - 业务冒烟（注册/登录/重置/改密/改邮/附件上传/靶机/题库导入等）。
- 建议扩展：富文本渲染 XSS 过滤测试、CSRF 场景（若启用跨域 Cookie）。

## 10. 处理策略
- 业务异常一律抛 BizError 子类，避免暴露栈信息；基础设施异常映射 503xx。
- 触发限流/风控返回 42900（RateLimitError），携带 wait/原因。
- 统一响应格式由 `apps.common.response` 提供。
